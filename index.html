ch.py

Distributed under the terms of the GNU GPL.

An event-based library for connecting to one or more Chatango chat rooms. Has support for several things including: messaging, message font, name colour, deleting, banning, recent history, two userlist modes, flagging, avoiding flood bans, and detecting flags.

Easy Creation of Bots

With ch.py, creation of Chatango bots is made easy. A functional bot can even be made in less than ten lines of Python code!

#!/usr/bin/python
import ch
class TestBot(ch.RoomManager):
  def onMessage(self, room, user, message):
    self.safePrint(user.name + ': ' + message.body)
    if message.body.startswith("!scream"):
      room.message("AAAAAAAAAAAAAA")
if __name__ == "__main__":
  TestBot.easy_start()
